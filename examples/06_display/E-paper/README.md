# Pin assignments

The table below shows the E-paper display pin assignments.

E-paper display Pin      | SPI pin | DShanMCU-Mio pin     |  Notes
-------------------------|---------|----------------------|-------------
 BUSY                    |         | GPIO7                |
 RESET                   |         | GPIO6                |
 D/C                     |         | GPIO5                |
 CS                      | CS      | GPIO10               |
 SCLK                    | SCK     | GPIO12               |
 SDI                     | MOSI    | GPIO11               |
