# Pin assignments

The table below shows the MicroSD Card Adapter pin assignments.

MicroSD Card Adapter pin | SPI pin | DShanMCU-Mio pin     |  Notes
-------------------------|---------|----------------------|-------------
 CS                      | CS      | GPIO15               |
 SCK                     | SCK     | GPIO16               |
 MOSI                    | MOSI    | GPIO17               |
 MISO                    | MISO    | GPIO18               |
